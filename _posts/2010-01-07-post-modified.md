Network Traffic Analysis is the process of capturing, monitoring, and network data. In this post we'll explore **Wireshark** and **tcp dump**

## wirehsark
This is a graphical packet analysis tool that allows users to capture and inspect network traffic in real time.

## tcpdump
This is a command-line packet capture tool used mainly in Linux and Unix systems.
